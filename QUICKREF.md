## Espresso Dial-In Cheat Sheet

| Parameter | Start Point | Target |
|-----------|------------|--------|
| Dose | 18g in | 18-20g |
| Yield | 36g out | 36-40g (1:1.8-2:1) |
| Time | 25-30 sec | 25-35 sec |
| Grind | Mid-fine | Adjust 1 notch at a time |

**Ratio guide:** 1:1.8 = ristretto, 1:2 = standard, 1:2.5 = lungo

**Order of adjustment:** Dose → Yield → Grind → Tamp (never tamp twice)

## Supply Chain Basics

| Term | Meaning |
|------|---------|
| **FOB** | Buyer owns goods at port of origin |
| **CIF** | Seller pays freight/insurance to destination |
| **MOQ** | Minimum Order Quantity (typically 1-5 bags of 60kg) |
| **30kg GrainPro** | Standard specialty bag size |
| **Container** | 20ft = ~275 x 60kg bags, 40ft = ~550 x 60kg bags |
| **Lead time** | 4-6 weeks sea freight, 1-2 weeks air |

## C-Market vs Specialty

- **C-market** — commodity coffee price (ICE Futures), ~$3-5/lb
- **Specialty premium** — +$0.50-$3/lb above C-market for 84+ lots
- **Competition lots** — $20-$100+/lb (auction)
- **Geisha** — $50-$500+/lb (Panama)

## Pricing Formula (Green Coffee)

```
Cost per kg   = (Bag price per lb ÷ 2.205 lb/kg)
Cost per g    = Cost per kg ÷ 1000
Cost per shot = Dose(g) × Cost per g
```

Example: 60kg bag at $13/lb FOB
- $13/lb ÷ 2.205 = $5.90/kg
- $5.90/kg ÷ 1000 = **~$0.0059/g**
- At 18g dose = **~$0.11 per shot** (just green coffee cost)

> ⚠️ C-market prices are volatile. Check current Arabica futures before any pricing or contract decision.

## Workflow Order

1. **Source** → Request samples → Cup → Select lot
2. **Contract** → Negotiate FOB → Agree QC clause → Place order
3. **Ship** → Book container → Track → await arrival
4. **Receive** → QC check against contract → Approve warehouse
5. **Roast** → Profile development → Lock in curve → Production
6. **QC** → Cup every production roast (Cuppers Checklist)
7. **Package** → Nitrogen flush, valve bag, label
8. **Ship** → Fulfillment → Customer delivery

## Common Defects (SCA)

| Defect | Effect | Tasting Note |
|--------|--------|-------------|
| **Full black** | 1 defect per 100g | Burnt, smoky, bitter |
| **Sour** | 1 per 100g | Fermented, vinegar |
| **Dried cherry** | 1 per 100g | Winey, fermented |
| **Insect damage** | 1 per 100g | Musty, earthy |
| **Shell** | 2 per 300g | Hollow, papery |

Primary defects: 0 = clean, 1-4 = specialty possible, 5+ = below specialty

## Key Temperature Targets

| Stage | Target |
|-------|--------|
| Charge temp | 180-200°C (356-392°F) |
| First crack | 196-204°C (385-400°F) |
| Development start | Post FC + 30-60 sec |
| Drop temp | 210-220°C (410-428°F) |
| Rest before roast | Green must rest 24h+ after arrival |
